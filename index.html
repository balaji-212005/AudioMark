Hide an encrypted logo invisibly inside any audio file using DWT-based Quantisation Index Modulation with Reed-Solomon error correction.

Quick Start

# 1. Install dependencies
pip install flask flask-cors reedsolo pydub pywavelets numpy Pillow cryptography matplotlib

# 2. Place all four files in the same folder:
  app.py        ← Flask backend
  index.html    ← this page
  embed.py     ← your existing embed
  extract.py   ← your existing extract

# 3. Start the server
python app.py

# 4. Open in browser
http://localhost:5000

How it works

Browser  ──POST /embed──►  app.py  ──calls──►  embed.embed()
                                                    │
                                              embed.py (unchanged)
                                                    │
         ◄── watermarked audio ──────────────────────┘

Browser  ──POST /extract──►  app.py  ──calls──►  extract.extract()
                                                      │
                                              extract.py (unchanged)
                                                      │
         ◄── recovered logo ────────────────────────────┘
